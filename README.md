# Nvidia Isaac Mission Dispatch Sandbox/Development Environment

## Codespace

### Getting started

To get started with Nvidia Isaac Mission Dispatch sandbox, run the following command to pull and run the Nvidia Isaac containers from NGC:

```bash
docker-compose up -d
```
